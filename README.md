# Lull Match Legal Pages

This repository contains the public legal pages for **Lull Match**.

## Pages
- **Privacy Policy**
- **Terms of Service**

These files are published through **GitHub Pages** and are used for:
- Google Play Console
- in-app legal links
- public access to app legal information

- `https://jolybin.github.io/lull-match-legal/`

- `https://jolybin.github.io/lull-match-legal/terms.html`

## Repository Purpose
This repository is used only for hosting legal documents for the app.

It does **not** contain the source code of the game.
